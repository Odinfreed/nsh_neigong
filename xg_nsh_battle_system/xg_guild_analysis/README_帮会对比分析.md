# 帮会对比分析系统 - 使用说明

## 系统简介

帮会对比分析系统为《逆水寒》帮会帮战提供全面的数据分析功能，支持两个帮会的数据对比、排行榜统计、个人分析、职业对比等功能。

---

## 文件清单

```
xg_guild_analysis/
├── xg_guild_analysis_main.html      # 主分析页面（完整功能）
├── xg_guild_analysis_backup.html    # 备份页面（简化功能）
└── README_帮会对比分析.md           # 本文件
```

## 核心功能

### 1. 全局排行

**榜单类型**：
- 🔥 伤害榜（对玩家伤害 / 对建筑伤害）
- 💀 击败榜
- ❤️ 治疗榜
- 🛡️ 卸甲榜（人伤卸甲 / 破塔卸甲）

**显示内容**：
- 排名（前3名高亮显示）
- 玩家名称
- 隶属帮会（带颜色区分：帮会1红色/帮会2蓝色）
- 职业
- 数值

### 2. 核心统计

**对比指标**：
- 成员数量
- 击败/助攻/资源
- 对玩家伤害/对建筑伤害
- 治疗值
- 人伤卸甲/破塔卸甲
- 化羽/清泉/焚骨

**战力评分**：
- 综合评分计算
- 各指标权重分析
- 优势方判定

### 3. 数据图表

**图表类型**：
- 📈 职业分布（折线统计图）
- 📊 伤害对比（柱状图）
- 📊 治疗对比（柱状图）
- 📊 击杀辅助对比（柱状图）
- 🕸️ 综合实力雷达图（总数/平均对比）

**样式特点**：
- 折线图使用直线连接（非曲线）
- 柱状图使用直角（非圆角）
- 颜色主题：帮会1红色、帮会2蓝色

### 4. 对比分析

**对比维度**：
- 成员数量对比
- 击败/助攻对比
- 治疗值对比
- 伤害输出对比

**可视化**：
- 进度条对比
- 数值差距显示
- 百分比优势计算

### 5. 个人分析

**玩家选择**：
- 🔍 按名称搜索
- 🏛️ 按帮会筛选（全部/帮会1/帮会2）
- 🎯 按职业筛选

**分析内容**：
- 基础数据展示
- 伤害与治疗雷达图
- 战斗数据雷达图
- 本帮会本职业排名
- 隶属帮会显示

**全屏详细分析** 🆕
- 📺 点击"全屏"按钮进入沉浸式分析视图
- 📊 **4图并排显示**：伤害雷达图 + 战斗雷达图 + 职业伤害对比 + 职业战斗对比
- 📈 多维度详细数据卡片
- 🖼️ 一键保存为图片（自动包含玩家名称、职业、帮会信息）
- 💾 图片命名格式：`玩家名字_详细分析_日期.png`
- 📉 大数值智能简化（自动转换为万/百万单位）

### 6. 职业对比

**对比图表**：
- 伤害与治疗雷达图
- 战斗数据雷达图

**数据维度**：
- 各职业平均数据
- 玩家数据vs帮会平均

### 7. 职业末位检测 🆕

**功能说明**：
- 🔒 隐藏功能，需输入特定指令激活
- 📊 检测帮会成员在全职业中的排名位置
- ⚠️ 显示帮会1中处于全职业后20%的成员
- 🏆 显示帮会2中处于全职业前20%的成员

**激活方式**：
1. 切换到"个人分析"标签页
2. 在搜索框输入 `OX208`
3. 自动显示"职业末位"标签并切换视图

**显示内容**：
- 按职业分组显示
- 标记排名靠后的指标
- 帮会名称动态同步

## 使用说明

### 打开分析页面

1. 打开 `xg_guild_analysis_main.html`
2. 系统自动加载已保存的数据
3. 如没有数据，可点击"加载示例数据"

### 修改帮会名称

**双击编辑**：
1. 找到页面顶部的帮会标签
2. **双击**帮会名称标签
3. 输入新名称并确认
4. 所有图表和榜单自动更新

**名称缓存** 🆕
- 修改后的帮会名称自动保存到浏览器缓存
- 刷新页面后自动恢复上次设置的名称
- 无需重复修改，提升使用体验

### 查看排行榜

1. 切换到"全局排行"标签页
2. 点击顶部分类按钮切换榜单
3. 查看TOP20排名
4. 帮会名称带颜色区分

### 个人分析

1. 切换到"个人分析"标签页
2. 使用筛选按钮选择帮会（可选）
3. 使用职业按钮筛选职业（可选）
4. 从下拉列表选择玩家
5. 查看详细分析和排名

### 全屏详细分析与截图

1. 在"个人分析"页面选择玩家后，点击"全屏"按钮
2. 进入全屏分析视图，查看大屏雷达图和详细数据
3. 点击"保存为图片"按钮生成分析截图
4. 截图自动包含：
   - 玩家游戏ID
   - 职业信息
   - 所属帮会（红色/蓝色区分）
   - 完整数据分析卡片
   - 雷达图

## 技术支持

### 常见问题

**Q1: 排行榜数据不更新怎么办？**

**A**: 
1. 检查是否已导入数据
2. 尝试点击"刷新数据"按钮
3. 查看浏览器控制台是否有错误

**Q2: 帮会名称修改后图表颜色没变？**

**A**: 
1. 确认已成功保存新名称
2. 切换到其他标签页再返回
3. 刷新页面重新加载

**Q3: 个人分析找不到某个玩家？**

**A**:
1. 检查"帮会筛选"是否选择了正确帮会
2. 尝试清除搜索框内容
3. 确认玩家数据已正确导入

**Q4: 图表显示异常？**

**A**:
1. 检查浏览器是否支持Canvas
2. 确认Chart.js库已正确加载
3. 查看控制台是否有脚本错误

**Q5: 截图保存失败？**

**A**:
1. 检查浏览器是否允许下载
2. 尝试关闭弹出窗口拦截器
3. 确保页面完全加载后再截图
4. 如遇到"createPattern"错误，请刷新页面重试

**Q6: 截图中信息不完整？**

**A**:
1. 确保已进入全屏分析模式
2. 检查是否已选择玩家
3. 截图会自动包含玩家名称、职业、帮会信息
4. 如仍有问题，请清除浏览器缓存后重试

**Q7: 如何查看职业末位检测功能？**

**A**:
1. 切换到"个人分析"标签页
2. 在搜索框输入 `OX208`
3. 系统会自动显示"职业末位"标签并切换到该视图
4. 输入框清空后标签会自动隐藏

**Q8: 帮会名称修改后刷新页面又变回默认？**

**A**:
1. 确认浏览器允许使用localStorage
2. 检查浏览器隐私设置是否阻止数据保存
3. 帮会名称应自动保存到浏览器缓存
4. 如仍有问题，尝试清除缓存后重新设置

**Q9: 职业对比图表数值显示异常？**

**A**:
1. 系统会自动将大数值转换为万/百万单位
2. 例如：1500000 显示为 1.5M 或 150w
3. 鼠标悬停可查看详细数值
4. 这是正常的数值简化显示，非错误
